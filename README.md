KaufmannPro ist eine ASP.NET Core Razor Pages Anwendung zur Entwicklung eines ERP-Systems.

Design- und UI-Entscheidungen erfolgen ausschließlich auf Basis von DevExtreme-Komponenten.</br>
Eigene CSS-Overrides oder alternative UI-Frameworks werden bewusst vermieden.

Die Anwendung ist auf eine deutschsprachige Benutzeroberfläche (de-DE) ausgelegt,
bietet eine öffentliche Login-Seite und verwendet MariaDB als Datenbanksystem.
